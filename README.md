# diff-src
Output a Code Diff to PDF

## Dependencies

1. TCL >= 8.6
1. TexLive
1. LatexDiff >= 1.3

## Run

`tclsh diff-src.tcl -i ./example/sample-sources -o ./example/output.pdf`

Or:

First `chmod +x` the `diff-src.tcl` script.  Then run `diff-src.tcl -i ./example/sample-sources -o ./example/output.pdf`
